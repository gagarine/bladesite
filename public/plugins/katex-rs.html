KaTeX - Rust

Render math formulas using KaTeX - written in Rust

Blades natively supports rendering LaTeX formulas into MathML. Sadly, Chromium-based browsers don’t support it. This plugin renders the formulas into HTML using KaTeX, run with the QuickJS enginge. Formulas delimited by $ are rendered in inline mode and by $$ in display mode.

This plugin can be installed as

cargo install blades-katex

Then, it can be used in Blades as

[plugins.content]
katex = "blades-katex"

On each page where KaTeX should be rendered, it can be enabled by

plugins = ["katex"]
